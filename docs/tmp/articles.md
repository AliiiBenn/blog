# SEO Article Structure

## Article Core Components

Each article will have mandatory SEO-related fields to ensure optimal search engine visibility and user experience.

### Required Fields

#### Title
- **Purpose**: Primary on-page SEO factor and user attraction
- **Length**: 50-60 characters ideal
- **Format**: Include primary keyword near beginning
- **Style**: Compelling, action-oriented, or curiosity-driven
- **Naming**: Clear indication of content topic
- **Uniqueness**: No duplicate titles across the platform

#### Description
- **Purpose**: Meta description for search engine snippets
- **Length**: 150-160 characters maximum
- **Format**: Complete sentence with clear value proposition
- **Keywords**: Include 2-3 relevant keywords naturally
- **CTA**: Include implicit call to action or curiosity element
- **Uniqueness**: Generate unique descriptions for each article

#### Content
- **Purpose**: Primary body text providing user value
- **Structure**: Optimized for both readability and SEO
- **Length**: Minimum 1000 words for comprehensive coverage
- **Formatting**: Strategic use of headers, lists, and emphasis
- **Readability**: Grade 8-10 reading level for accessibility
- **Value**: Authoritative, informative content solving specific problems

### Technical SEO Elements

#### Meta Information
- **Canonical URL**: Automatic generation for each article
- **Open Graph tags**: Social media sharing optimization
- **Twitter cards**: Enhanced Twitter sharing experience
- **Schema markup**: Article structured data implementation
- **Language attribute**: Proper language declaration (en-US)

#### URL Structure
- **Format**: `/article/slug-optimized`
- **Keywords**: Include primary keywords in URL
- **Length**: Preferably 100 characters or less
- **Readability**: Human-friendly, hyphen-separated
- **Uniqueness**: Permanent URL generation on publish

## Content Organization

### Header Hierarchy
- **H1**: Article title (once per article)
- **H2**: Main sections and key topics
- **H3**: Sub-sections and supporting points
- **H4**: Detailed explanations (when necessary)
- **Distribution**: Follow traditional inverted pyramid structure

### Content Density Requirements
- **Keyword density**: 1-3% for primary keywords
- **Content length**: Comprehensive coverage (1500+ words ideal)
- **Readability**: Mix of short and long paragraphs
- **Visual elements**: Image, charts, or code blocks every 300-400 words

### Internal Linking Strategy
- **Contextual links**: 3-5 relevant internal links per article
- **Anchor text**: Natural keyword variations
- **Link placement**: Strategically within content body
- **Link diversity**: Combination of new and existing content links

## Image SEO

### Image Requirements
- **File naming**: Descriptive, keyword-rich file names
- **Alt text**: Detailed description including keywords
- **File size**: Optimized for loading (under 100KB)
- **Format**: WebP preferred with fallbacks
- **Dimensions**: Responsive sizing with appropriate aspect ratios

### Image Content
- **Relevance**: Directly related to article content
- **Quality**: High-resolution, professional appearance
- **Originality**: Unique images when possible
- **Placement**: Strategic placement near related content
- **Captions**: Descriptive text when applicable

## Content Quality Standards

### Authoritative Content
- **Expertise**: Demonstrates knowledge and authority
- **Accuracy**: Well-researched, fact-checked information
- **Completeness**: Comprehensive coverage of topic
- **Up-to-date**: Current information where applicable
- **Sources**: Credible references when necessary

### User Experience Factors
- **Mobile optimization**: Responsive design guaranteed
- **Page speed**: Optimal loading times (<3 seconds)
- **Readability**: Clear typography and sufficient contrast
- **Navigation**: Logical structure and easy scanning
- **Value**: Immediate and clear user benefit

## SEO Metadata Implementation

### Search Engine Optimization
- **Title tags**: Dynamic page titles with separators
- **Meta descriptions**: Compelling summaries under 160 characters
- **Header tags**: Proper H1, H2, H3 structure implementation
- **Keyword placement**: Natural keyword integration throughout content
- **ALT attributes**: Comprehensive image alt text implementation

### Social Media Optimization
- **Open Graph**: Title, description, image, and URL optimization
- **Twitter Cards**: Enhanced social media sharing experience
- **LinkedIn sharing**: Professional network compatibility
- **Facebook sharing**: Social platform optimization
- **Pinterest compatibility**: Image-friendly format

## Performance Considerations

### Loading Speed
- **Image optimization**: Compressed images with proper formatting
- **Lazy loading**: Images load only when visible
- **Code optimization**: Minified CSS and JavaScript
- **CDN integration**: Fast content delivery network
- **Caching**: Browser and server-side caching strategies

### Mobile Optimization
- **Responsive design**: Mobile-first approach implementation
- **Touch-friendly**: Appropriate button and link sizing
- **Loading speed**: Optimized for mobile networks
- **Compatibility**: Cross-device and cross-browser support
- **Core Web Vitals**: Achievement of good performance scores

## SEO Best Practices

### Content Creation Guidelines
- **Keyword research**: Thorough keyword analysis before writing
- **Intent matching**: Align content with user search intent
- **Competitive analysis**: Understand existing content landscape
- **Unique value**: Provide superior content compared to competitors
- **Freshness**: Regular content updates where relevant

### Technical Implementation
- **XML sitemaps**: Automatic generation of sitemap entries
- **Robots.txt**: Proper guidance for search engine crawling
- **Canonical tags**: Prevention of duplicate content issues
- **Structured data**: Implementation of relevant schema markup
- **Analytics setup**: Comprehensive tracking and monitoring

## Quality Control Process

### Pre-Publish Checklist
- **Title validation**: Check length and keyword inclusion
- **Description verification**: Ensure optimal length and compelling content
- **Content review**: Comprehensive reading for accuracy and completeness
- **Image verification**: All images properly optimized and described
- **Internal links**: Confirm relevant internal links are present

### Post-Publish Monitoring
- **Performance tracking**: Monitor ranking and traffic trends
- **User engagement**: Analyze bounce rate and time on page
- **Content updates**: Regular review and refresh cycle
- **Technical SEO**: Continuously monitor technical performance
- **Competitive monitoring**: Track competitor content strategies
